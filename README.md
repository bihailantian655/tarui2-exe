# BAT_R - 批处理脚本管理器

BAT_R 是一个基于 Tauri 构建的桌面应用，用于管理和运行 `.bat` 批处理脚本。

## 功能

- **脚本管理**：添加、编辑、删除、复制、重命名 `.bat` 脚本
- **一键运行**：选中脚本后点击运行，无需打开命令行
- **拖拽排序**：拖拽配置项调整顺序
- **脚本编辑**：内置脚本编辑器，可直接修改脚本内容
- **主题切换**：支持白天/夜间/自动切换模式

## 下载安装

前往 [Releases](https://github.com/bihailantian655/tarui2-exe/releases) 页面下载最新版本的 `bat-r2.exe`，双击运行即可。

## 使用说明

### 添加配置

1. 点击右上角 **+** 按钮
2. 输入配置名称和文件夹路径
3. 点击 **确认添加**

### 编辑配置

1. 点击配置项上的 **编辑** 按钮
2. 在弹窗中输入文件夹路径，点击 **刷新** 加载 `.bat` 文件列表
3. 选择要关联的脚本，可修改脚本内容
4. 点击 **确认修改** 保存

### 运行脚本

- 点击配置项上的 **运行** 按钮，直接运行关联的脚本
- 选中配置项后，使用顶部 **开始/停止** 按钮控制脚本运行

### 其他操作

- **复制**：复制当前配置项
- **上移/下移**：调整配置项顺序
- **删除**：删除配置项

## 开发

### 环境要求

- Node.js 20+
- Rust (stable)
- Tauri CLI

### 本地运行

```bash
npm install
npm run dev
```

### 构建

```bash
npm run build
npx tauri build
```

## 技术栈

- [Tauri 2](https://tauri.app/) - 桌面应用框架
- HTML / CSS / JavaScript - 前端界面
- Rust - 后端逻辑
